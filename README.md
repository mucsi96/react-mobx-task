# react-mobx-task

## Technical stack

- React
- Mobx
- TypeScript
- Jest + react-testing-library
- Parcel
- Semantic-UI
- TSLint
- Prettier

## NPM scripts

- `start` starts development server (Parcel)
- `test` run tests (Jest)
- `test:watch` run tests in watch mode (Jest)
- `build` create production build (Parcel)
- `lint` lints the codebase (ESLint)
